# PokeView-App

A Pokédex application that displays a list and details of Pokémon by loading data from an external API.

## Context

This is app is an exercise to learn Front End Development using HTML, CSS and Javascript.

## Things used

- jQuery
- Bootstrap
- Visual Studio Code
- Pokemon API
